June 15 - June 20 Learning Review

While I was working this week I did not accomplish what I had hoped to. I didn’t work as much as I had anticipated for a few reasons but they are not important; I still failed to accomplish a majority of these tasks.

My intended focuses:

  • Update Passwords
  • Start Unreal Engine Learning
  • Windows Exploit Development Course Review Post

Repeating Activities include:

  • Read and complete PFsense Book/Course
  • Study for PCNSA Exam
  • Start Active Directory Learning
  • HTB (Retired) Walkthroughs
  • Apply for Jobs

Additional Non-Focused Activities:

  • Improve OpenVPN Autodeployment script (v1.2)
  • Security Onion Research

Review

While I did finish more than last week I still am not proud of the amount I accomplished. Most of the tasks that are vague like “continue” means any length of participation in that task does count toward it. On said tasks I did very little so I don’t count them as an accomplishment. I am happy that I was able to update my passwords (only took a year) and study for the PCNSA. I did move my test date back due to my late start but I’m excited to continue the course. Most of the topics on here will rollover, but I plan on adopting a new philosophy to study here momentarily so my hope is that I can have more efficient time to study.

Takeaway

Adopt a new philosophy so I can be more productive.
