June 22-June-28 Learning

The name of the game this week is finish the tasks. I’ll be busy, but I have a new philosophy in place I plan to apply to boost my productivity and actually accomplish these topics. We’ll see what comes this week but I do intend to finish everything on the list. Here is what I’ll be up to.

My intended focuses:

  • Start Unreal Engine Learning
  • Windows Exploit Development Course Review Post

Repeating Activities include:

  • Read and complete PFsense Book/Course
  • Study for PCNSA Exam
  • Start Active Directory Learning
  • HTB (Retired) Walkthroughs
  • Apply for Jobs

Additional Non-Focused Activities:

  • Improve OpenVPN Autodeployment script (v1.2)
  • Security Onion Research

Challenges and Intentions

I actually plan on finish everything this week. The Windows Exploit Development Post I plan to write today as well as continue studying for the PCNSA which I pushed back the exam date for. The PFSense is a bit off lower on the scale of priorities as I already am studying a firewall but if I have spare time I’ll focus on doing that. I do need to start Active Directory learning and I’ll have no problem doing HTB walkthroughs. I’ll continue to Apply for Jobs and if I have time or need a distraction I’ll work on the additional activities.

This week’s challenges will include me being out of town again for a few days, having a few interviews, and working with my family on a series of personal projects. The most difficult things for me to complete will be starting Active Directory Learning, PFSense, and the non-focused activities. I also have other business that is not mentioned here that will take up some time.

This week does have a fair amount of topics (still less than usual) but nonetheless I’ll attempt them all. The focus is the PCNSA and Active Directory Learning, which you’ll see me work on throughout the week. Don’t forget to checkout my Trello Board to see what I am working on day-to-day.

Don’t wish me luck.
